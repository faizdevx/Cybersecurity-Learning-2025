# 🛠️ Cyber Attack Planning (Educational Flow)

This project demonstrates how attackers plan and execute attacks using a simplified cyber kill chain model. It includes:

- A visual attack planning flowchart (Mermaid.js)
- A safe, legal simulation of a port scan using Nmap on localhost

---

## 📁 Contents

- `attack_flowchart.md` – Visual kill chain
- `nmap-localhost-demo.md` – Reconnaissance simulation
- `screenshots/` – Sample output from Nmap
- `.drawio` – Optional visual flowchart file

---

## ⚠️ Disclaimer

This project is for **educational purposes only**. Never perform scanning or exploitation on any system you don’t own or have permission to test.
